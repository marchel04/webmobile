proyek

HTML

HTML adalah bahasa markup yang umumnya digunakan untuk menyusun bagian paragraf, heading, maupun link pada halaman web. Namun, meskipun susunannya seperti coding

CSS

Cascading Style Sheets atau CSS adalah bahasa pemrograman berbasis aturan yang dikembangkan oleh World Wide Web Consortium (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi untuk memformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut.

JAVA

Java adalah bahasa pemrograman yang berorientasi pada objek (object-oriented programming). Salah satu keunggulan Java adalah kemampuannya untuk digunakan di berbagai platform (multi-platform).
